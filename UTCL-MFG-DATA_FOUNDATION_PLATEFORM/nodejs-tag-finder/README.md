# Node.js Tag Finder

This project is a simple Node.js application that reads JavaScript files from a specified directory, extracts text labeled as "tags," and outputs a unique list of these tags.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)
- [License](#license)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/nodejs-tag-finder.git
   ```

2. Navigate to the project directory:
   ```
   cd nodejs-tag-finder
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

1. Ensure you have Node.js installed on your machine.

2. Place your JavaScript files in a directory.

3. Update the `src/index.js` file to specify the path to your directory.

4. Run the application:
   ```
   node src/index.js
   ```

5. The unique list of tags will be displayed in the console as well as a new excel sheet will be created.
